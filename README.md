# Python实现微信跳一跳外挂

## 讲解

[Python实现微信跳一跳外挂](https://www.orztu.com/post/wechat-jump/)

## 使用方法

1. 安卓手机打开USB调试
    打开USB调试方法，进入`设置`，找到`开发者选项`，打开并勾选`USB调试`；
    如果没有`开发者选项`，进入`关于手机`，连续点击`版本号`7次，即可开启`开发者选项`

3. 进入微信跳一跳并点`开始游戏`，运行`python helper.py`
